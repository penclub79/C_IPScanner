# C_IPScanner

C++ 기반으로 네트워크 장비 탐색 및 IP 관련 유틸리티 기능을 구현한 Windows 데스크톱 프로젝트입니다. MFC 스타일의 네이티브 애플리케이션 구조와 네트워크 제어 관련 코드를 함께 다룬 경험이 담겨 있습니다.

## 프로젝트 소개

저장소 구조를 보면 `IPScanUtil.sln`과 `IPScanUtil` 프로젝트가 포함되어 있고, 내부에는 `ARP`, `NetServerScan`, `IPChangeThread`, `UpgradeThreadHttp`, `ResolutionChangeDlg`, `OSDChangeDlg` 같은 파일이 존재합니다. 이를 바탕으로 단순 IP 조회를 넘어서 네트워크 스캔, 설정 변경, 장비 관리 기능을 포함한 Windows용 유틸리티 프로젝트로 볼 수 있습니다.
> C++ 기반 Windows 유틸리티 프로젝트로, 네트워크 장비 탐색과 IP 관련 제어 기능을 다뤄보며 네이티브 애플리케이션 구조와 네트워크 관련 로직을 함께 경험한 프로젝트입니다.

## 사용 기술

- C++
- Visual Studio Solution
- Windows 데스크톱 애플리케이션 구조
- 네트워크 관련 유틸리티 로직

## 주요 기능

- 네트워크 장비 스캔
- ARP 관련 처리
- IP 변경 관련 기능
- 연결 확인 스레드 처리
- 업그레이드/설정 변경 관련 대화상자 구조

## 제가 중점적으로 경험한 부분

- C++ 기반 프로젝트에서 기능별 클래스를 나누고 네트워크 관련 로직을 구성하는 방식을 익혔습니다.
- 스레드 처리와 다이얼로그 기반 UI가 함께 있는 Windows 유틸리티 구조를 다뤘습니다.
- 단순 알고리즘이 아니라 실제 장비 관리형 도구에 가까운 코드를 다뤄본 경험을 쌓았습니다.

## 실행 방법

Visual Studio에서 솔루션 파일을 열어 빌드 후 실행합니다.

예상 실행 절차:

1. `IP_Scanner/IPScanUtil.sln` 열기
2. Visual Studio에서 Debug 또는 Release 빌드
3. 실행 파일 생성 후 동작 확인

## 배운 점

- C++ 프로젝트는 기능 구현뿐 아니라 빌드 환경, 의존성, 운영체제 환경에 따라 개발 난이도가 높아진다는 점을 배웠습니다.
- 네트워크 유틸리티는 단순한 화면 구현보다도 스레드, 장비 상태, 설정 변경 로직을 안정적으로 다루는 것이 중요하다는 점을 경험했습니다.
